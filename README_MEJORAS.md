## 📱 **7. Integración y Uso**

### **Componentes Globales:**
```tsx

<SoundManager />
<AchievementSystem />
<ThemeToggle />
<TouchGestures />
```

### **Hooks Personalizados:**
```tsx

const { playButtonClick, playSuccess } = useSound()

// Hook para logros
```
